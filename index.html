Elements UI is a simple, modular and lightweight component library that gives you the building blocks to build React applications.

Installation

Elements UI components use emotion as their style engine.

To install Elements UI in a React project, run either of the following command depending on the package manager used in the project:

npm i -D @rishavbharti/elements-ui yarn add --dev @rishavbharti/elements-ui

Usage

Here's a quick example to get you started, it's literally all you need:

import React from 'react';

import ReactDOM from 'react-dom';

import { Button } from '@rishavbharti/elements-ui';


function App() {

return <Button variant="contained" />;

}

import React from 'react';

import ReactDOM from 'react-dom';

import { TextField } from '@rishavbharti/elements-ui';


function App() {

return <TextField helperText="Please enter your name" label="Name" required />;

}
